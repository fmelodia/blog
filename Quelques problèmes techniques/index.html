L’orientation des photos

Prises avec un téléphone, les photos sont toutes enregistrée dans un format paysage. Quand on prend une photo en format portrait, l’appareil ajoute dans les données de la photo une information « Orientation » qui indique comment l’image doit être affichée.

Problème : le navigateur ne prend pas en compte cette information. Il existe dans CSS une propriété image-orientation: from-image qui permet d’indiquer qu’il faut prendre en compte cette valeur.

Problème (bis) : le seul navigateur qui prend en compte cette propriété CSS pour le moment, c’est Firefox.


La seule solution que j’ai trouvée pour le moment, c’est de recadrer la photo avant de l’insérer dans Bear. Par exemple en format carré.
